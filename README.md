# elfalyzer - ELF file section size visualization

This python script is used to visualize section sizes and shows them as a memory map in the terminal. Your terminal should support UTF-8, if not see step four.

To visualize, do the following steps:
1. Add the sections and their size in `SECTION_SIZE_LIMITS`
2. Adjust `max_height` to properly scale to your terminal height
3. Adjust `width` to properly scale to your terminal width
4. If your terminal does not support UTF-8, change the block symbols to ASCII

An example C file and a linker script is included. To compile the C file:
```
$ gcc -o example.elf example.c -T example.ld -nostartfiles
```
The C file is not runnable and has a segmenation fault. It's main purpose is just to quickly show the functionality of elfalyzer.
